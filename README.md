# gameaward2016
ゲーム大賞2016応募作品　お題は「流」
## ゲームの趣旨
未完成の油時計にレールを置いて油時計を完成させるゲーム

## クリア条件
↓から出てくる油を反転した時、↑に一定数入ればクリア
## 操作方法
 - 画面下にあるアイコンをタップするとレールを生成することが可能
 - レールをすべて置く
 - レールは長押しすることで回転させることが可能
 - レールをすべて置いたら画面左の再生ボタンを押す
 - レールは重なっていると再生することが出来ない
 - 流れを楽しむ
